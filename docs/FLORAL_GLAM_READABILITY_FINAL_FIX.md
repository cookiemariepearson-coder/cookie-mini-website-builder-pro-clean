## What this fixes

The Floral Glam template was switching correctly, but the background was too bright and the words were hard to read.

This fix keeps the Floral Glam style, but improves readability by:

- darkening the text contrast
- adding a soft white text panel behind the hero wording
- keeping the pink/floral look
- keeping the purple/gold brand style
- keeping the template switch behavior working

## No SQL needed

This is only a CSS readability fix.

## Test after Vercel says Ready

1. Open Builder.
2. Choose Beauty / Hair / Salon.
3. Select Floral Glam.
4. Confirm the words are easy to read.
5. Switch to Luxury Salon and back to Floral Glam.
6. Confirm both looks still switch correctly.
