# Effect on Website Plans

The AI Video Studio $5 option is separate from website subscriptions.

Recommended current setup:
- Free Launch Page: $0, up to 3 selected sections, no included real AI video
- Starter Pro: $19/month, up to 4 selected sections + media options, no included real AI video
- Business: $30/month, up to 6 selected sections + media options + included AI Video Studio access
- Premium: $50/month, all built-in sections + media options + strongest AI Video Studio access
- Extra Page Add-On: $10/month per page
- AI Video Studio Only: $5 one-time standalone option

This keeps lower website plans affordable while still giving Video Genie / Instagram traffic a way to buy video help without needing a website.
