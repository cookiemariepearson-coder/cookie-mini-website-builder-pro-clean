# Cookie Mini Website Builder Pro - Action Selector + Bold Template Fix

Upload these inside files/folders to the clean GitHub repo only:

cookie-mini-website-builder-pro-clean

## What this fixes

### Order / Book / Buy selection
The wording was showing, but the customer could not clearly select the action section and action buttons.

This fix adds real clickable buttons directly under the guide:

- Select Order / Book / Buy
- Add Book Now
- Add Order Now
- Add Buy Now
- Add Request Quote

It also makes the Order / Book / Buy section button clickable even when the plan limit is full. If the plan is full, it swaps out the last selected non-home section so the action section can be added.

### Bold look / Bold font
The bold look was not visually changing enough.

This fix makes Bold noticeably different:
- Bigger uppercase hero headline
- Stronger bold section titles
- Gold shadow/highlight
- Stronger button styling
- Stronger Bold Sales visual look

### Template look switching
This also adds extra styling for:
- Bold Sales
- Creator Tool / 3D looks
- Cartoon looks
- Luxury looks
- Clean looks
- Cinematic looks

## No SQL needed

This is only a builder/template/CSS fix.
