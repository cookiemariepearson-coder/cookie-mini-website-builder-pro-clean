# Customer Action Upgrade Notes

This update adds conversion tools without building a full e-commerce system.

The platform does not process the website owner's customer orders, booking requests, deliveries, refunds, inventory, chargebacks, or disputes. Instead, the website owner can connect their own third-party links.

Examples:
- Gumroad product link
- Square payment link
- Stripe payment link
- Calendly booking link
- Google Forms order form
- Jotform order form
- Cash App link
- phone call
- SMS text
- email
- menu link
- social media DM link

This keeps launch simple and useful while avoiding a full shopping cart/inventory/order backend for now.
