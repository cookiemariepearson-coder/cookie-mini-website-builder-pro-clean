# Cookie AI Assistant v3 Setup

1. Upload the patch to:
cookie-mini-website-builder-pro-clean

2. In Vercel, confirm:
OPENAI_API_KEY = your private OpenAI API key

3. In Vercel, add/update:
OPENAI_MODEL = gpt-5.1

4. Redeploy.

5. Click Clear in Cookie AI before testing.

This version removes the hard-coded plan-state wizard and lets the smarter model handle the conversation naturally.
