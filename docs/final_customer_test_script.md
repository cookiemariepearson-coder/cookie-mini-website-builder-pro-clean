# Final Customer Test Script

1. Open https://www.cookiesdigitalcreations.com/builder
2. Choose Free Launch Page.
3. Pick a website type and template look.
4. Enter business name, email, headline, description, and offers.
5. Upload a hero image.
6. Add a gallery/media link if available.
7. Click Save Draft.
8. Open AI Video Studio and generate a kit.
9. Return to the builder.
10. Publish the free page.
11. Open the customer subdomain.
12. Go to /customer and search by email.
13. Continue/edit the draft or published site.
14. Go to /admin, enter PIN, check the record, add a note, and lock admin.
15. Test Starter, Business, Premium, and Extra Page checkout routes.
