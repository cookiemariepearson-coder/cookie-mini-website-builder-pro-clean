# Cookie Mini Website Builder Pro - Floral Glam Dark Readability Fix

Upload these inside files/folders to the clean GitHub repo only:

cookie-mini-website-builder-pro-clean

## What this fixes

The Floral Glam template was still too bright.

This fix makes Floral Glam a darker luxury floral look so the wording is much easier to read.

It changes Floral Glam to:

- dark purple hero background
- dark wording panel
- white hero headline
- light readable description text
- gold kicker text
- floral pink accents
- dark offer cards with white text

The template still keeps its floral/glam personality, but it should no longer wash out the words.

## No SQL needed

This is only a CSS readability fix.

## Test after Vercel says Ready

1. Open Builder.
2. Choose Beauty / Hair / Salon.
3. Select Floral Glam.
4. Confirm the headline and description are easy to read.
5. Switch to Luxury Salon and back to Floral Glam.
6. Confirm both looks still switch correctly.
