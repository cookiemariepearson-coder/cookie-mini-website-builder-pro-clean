# Cookie AI Assistant v1 Setup

1. Upload the files/folders into the clean repo:
cookie-mini-website-builder-pro-clean

2. In Vercel, add:
OPENAI_API_KEY

3. Optional:
OPENAI_MODEL = gpt-4o-mini

4. Redeploy Vercel.

5. Test:
https://www.cookiesdigitalcreations.com/

This first version only guides customers. It does not edit websites, publish websites, access admin, process refunds, or see private accounts.
