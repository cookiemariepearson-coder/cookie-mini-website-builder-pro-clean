# Gumroad Product Setup Checklist

For each product, check:

- Product title is correct.
- Product price is correct.
- Product image is added.
- Description is updated.
- Custom field/question is added when needed.
- Thank-you/content message is added.
- Return link is correct.
- Support/contact instructions are included.

## Custom field for paid products

Add this custom field/question to Starter, Business, Premium, and Extra Page Add-On:

Website name or subdomain you are upgrading

Helper text:

Example: cookies-kitchen-digital-recipes. Do not include https:// or .cookiesdigitalcreations.com.

Make this field required.
